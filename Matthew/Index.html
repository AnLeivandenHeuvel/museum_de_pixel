Super Mario Sunshine

Super Mario Sunshine is een 3D platformer game gemaakt door Nintendo. voor het gameplatform GameCube.
Super Mario Sunshine kan worden opgedeeld in drie genres:

  • Platformer Game
  • Adventure Game
  • Action Game

Geschiedenis van Super Mario Sunshine

Nintendo kondigde Super Mario Sunshine als eerste aan in 2001, tijdens Nintendo Space World,
ook kondigde Nintendo aan dat dit hun 2de 3D game zou worden. De eerste was Mario 64.
Later kregen mensen nog een kijkje naar Super Mario Sunshine tijdens E3 2002.


Het duurde 1.5 jaar voor Nintendo om de game volledig te ontwerpen, Nintendo fans waren
erg blij om een nieuwe 3D Mario Game te zien. Er werden ruim 5.91 miljoen exemplaren verkocht!
Tot en met de dag van vandaag word er best veel over deze game gepraat in de Mario community.
